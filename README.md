# Enterprise Hub

Enterprise Hub is a modular internal portal designed to centralize operational, administrative and automation processes within an organization.

The platform combines internal communication, resource management, document access, automation workflows and AI-assisted tools into a single web application.

## Features

### Administration

* User management
* Access control
* Application management

### Internal Communication

* Corporate news publishing
* Standard Operating Procedures (SOPs)
* Centralized information hub

### Resource Management

* Meeting room reservations
* Agenda management
* Internal resource scheduling

### IT Operations

* IT support requests
* Equipment inventory management

### Human Resources

* HR module
* Employee-related processes

### Finance & Fiscal

* Fiscal dashboard
* Administrative workflows

### Automation

* RPA generation tools
* Autonomous workflow execution

### Artificial Intelligence

* Local AI integration through Ollama
* Internal assistant support
* Automated task assistance

## Technology Stack

* Python
* Flask
* SQLite / Database Layer
* HTML
* CSS
* JavaScript
* Ollama

## Project Structure

The application follows a modular architecture where each business area is isolated into its own module, allowing independent development and maintenance.

## Purpose

This project was developed to explore enterprise application architecture, process automation, internal operations management and local AI integration in a unified platform.
